# Design-md-ai
Figma to Markdown | Scan components &amp; variables • Export .md files • AI Design Assistant | Secure &amp; Fast
1. Create .md Project Files
Easily create and manage comprehensive Markdown project files with advanced Figma scanning capabilities.
Key Capabilities:

Figma Scan Projects: Automatically extract detailed file statistics including totals, pages, variables, files, tokens, and UI points.
Project Creation & Customization: Create new projects by renaming files or editing metadata such as categories, types, language models, design styles, pre-installed Open Design systems, and layout templates.
Template Mapping: Configure flexible template maps for automatic or manual component addition.
Project Explorer:
View complete folder structure
Inspect variables and design tokens

Quick Actions: One-click buttons for downloading files, exporting AI frames, and copying content.

2. Integrated Channel Design Tool
A powerful built-in design-to-code assistant that generates standard .md files aligned with your design system.
Highlights:

Includes the latest .md component libraries and design tokens.
Supports file uploads, AI image creation, image uploads, and direct output of Markdown files or clean display code.
Acts as an intelligent assistant for exporting production-ready .md files.
Helps shape your product according to defined workflows and scenarios based on open-source best practices.
