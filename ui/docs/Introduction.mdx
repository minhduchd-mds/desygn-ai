# Desygn AI Component Library

UI components for **Desygn AI v5** — the agentic UI/UX auditor connecting Figma to AI coding agents.

> 📦 [GitHub](https://github.com/minhduchd-mds/desygn-ai) · 🌐 [Live App](https://design-md-ai-yd6r.vercel.app) · 📖 [Docs](https://github.com/minhduchd-mds/desygn-ai/tree/main/docs)

| Surface | Stories location |
|---------|------------------|

| **Plugin SDK** — coming soon | Plugin/* |

## What Desygn AI does

1. **Scans** Figma designs across 6 quality dimensions (naming, structure, tokens, meta, completeness, variants)
2. **Audits** UI/UX with 8 specialized AI agents (Design Audit, Accessibility, Design System, Score, Recommend, Fix Planner, Issue Writer, Memory)
3. **Self-learns** with HNSW vector search + sigmoid decay across projects
4. **Integrates** with GitHub (issues, PRs, CI gate, deploy gate)
5. **Streams** real-time audit events at 60fps

## Components

| Component | Purpose |
|-----------|---------|
| **ScoreOverview** | Donut chart showing the AI-Readiness score across 6 dimensions |
| **AutoLayoutFix** | One-click Auto Layout conversion for frames |
| **FixPanel** | Rename layers, convert dividers, delete hidden elements |
| **AtomicBadge** | Atom / Molecule / Organism classification |
| **PromptExport** | Copy the generated prompt with token estimate |
| **TokenMap** | Click-to-locate color token coverage |
| **BatchPanel** | Multi-component scan with atomic build order |
| **ProfileManager** | Design System Profiles (tokens, stack, guidelines) |
